Local Setup Guide
1 - Install Dependencies
Create a virtual environment and install Python packages.
python -m venv venv
source venv/bin/activate
# Windows: venv\Scripts\activate
pip install -r requirements.txt
playwright install chromium
2 - Configure Email
Edit config.yaml. Use a Gmail App Password (Google Account > Security > 2FA > App Passwords).
email:
sender: "[email protected]"
password: "xxxx xxxx xxxx xxxx"
recipient: "[email protected]"
3 - Run Locally
Two terminals: backend API + test scraper. Open the dashboard HTML in your browser.
# Terminal 1
python api.py
 
# Terminal 2
python main.py --dry-run
4 - GitHub Actions (Free)
Push to a private repo, add 3 secrets, runs every 4 hours automatically.
# Settings > Secrets > Actions
EMAIL_SENDER [email protected]
EMAIL_PASSWORD app-password
EMAIL_RECIPIENT [email protected]
